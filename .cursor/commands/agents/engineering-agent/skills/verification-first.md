# Skill: Verification First

## Objetivo

Garantizar que toda entrega cierre con validación reproducible.

## Reglas

- Siempre incluir comandos ejecutables
- Siempre incluir checklist final
- Priorizar tests del área modificada
- Reportar brevemente limitaciones de validación si existen

## Salida

Bloque de comandos + checklist listo para ejecutar en CI/local.
