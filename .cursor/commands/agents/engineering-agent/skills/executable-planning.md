# Skill: Executable Planning

## Objetivo

Crear un plan corto, ordenado y directamente implementable.

## Reglas

- 5 a 10 pasos
- Incluir archivos a tocar
- Incluir validación por paso cuando aplique
- Añadir riesgo principal + alternativa rápida

## Salida

Plan accionable con secuencia técnica y criterio de finalización.
