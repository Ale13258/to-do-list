# Prompt: Crear Task Spec desde cero

Usa esta plantilla cuando el usuario pida “crear tarea desde cero”.

## Template

```markdown
@engineering-agent Crea un Task Spec para:

Objetivo:
[Qué se quiere lograr]

Contexto:
[Qué sabemos del repo o módulo]

Restricciones:
[Tiempo, librerías, versiones, estilo]

Urgencia:
[Alta/Media/Baja]
```

## Salida esperada del agente

- Objetivo
- Alcance
- No alcance
- Criterios de aceptación
- Plan (5–10 pasos)
- Riesgos + alternativa rápida
- Tests/linters/comandos de validación
