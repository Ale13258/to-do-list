# Prompt: Fast MVP

Plantilla para pedir la vía más rápida con riesgo controlado.

## Template

```markdown
@engineering-agent Necesito la solución más rápida para:
[problema]

Condiciones:

- Entrega MVP primero
- Máximo cambio necesario
- No refactors grandes
- Incluye diff + comandos + checklist
```

## Criterio

El agente debe:

1. Proponer una solución mínima implementable.
2. Explicitar supuestos.
3. Dejar mejoras no críticas como **Next steps**.
